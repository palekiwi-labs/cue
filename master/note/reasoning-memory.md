From a presentation on using graphs DBs for memory:

```md
# Reasoning Memory

Decision traces, tool usage audits, and provenance - the layer that makes AI explainable

## 1. Tool call traces

Every tool invocation, parameter, results - complete audit trail

## 2. Decision provenance

Why did the agent choose this path? Causal chain fully recorded

## 3. Learning from experience

Agent checks if it solved something similar before, reuses successful patterns

## 4. Compliance and debugging

When something goes wrong you can trace exactly what happened and why.
```
